# 🚀 Personal Portfolio Website

A modern, responsive personal portfolio website built with **FastAPI**, **HTML5**, **CSS3**, and **vanilla JavaScript**. Showcases my projects, skills, and provides a way to get in touch — all served through a lightweight Python backend.

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![FastAPI](https://img.shields.io/badge/FastAPI-0.110-009688.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

---

## ✨ Features
- ⚡ FastAPI backend with Jinja2 templating
- 🎨 Clean, modern dark-themed UI
- 📱 Fully responsive design (mobile-friendly)
- 🖱️ Smooth scroll animations using Intersection Observer
- 📬 Working contact form with backend route handling
- 🗂️ Organized project & skills showcase sections

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | FastAPI, Jinja2 |
| Frontend | HTML5, CSS3, JavaScript |
| Server | Uvicorn |

## ⚙️ Getting Started

\`\`\`bash
git clone https://github.com/yourusername/portfolio-website.git
cd portfolio-website
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
\`\`\`

Visit **http://localhost:8000** 🎉
